ABOUT關於我（占位版 · 之後會用 Section SDD 替換）

這裡是 About 區的占位內容，主要用來簡單說明未來會在這裡放什麼樣的介紹文字。

正式版本的 About 區，預計會放上你的簡短背景介紹、學習或工作經歷重點、對前端開發的興趣與目前專注的方向。你可以使用 Section SDD 的 Prompt， 針對「關於我」這個區塊產生更完整的結構與文案。

產生完成後，請把新的 About section 程式碼（包含 <section> 標籤本身）， 完整貼在 SECTION:ABOUT-START / SECTION:ABOUT-END 註解中間， 完全覆蓋現在這一段占位內容。

給開發者說明：目前這裡只是示意版，用來維持版面結構與 RWD 行為。之後若需要調整排版或欄位數量，可以在新的 Section SDD 結果裡同時定義 HTML 與對應 CSS， 並以 class 為主來覆寫樣式。

Section HTML 插槽使用方式（About 區）
  • 1. 使用 Section SDD Prompt 產生新的 About 區內容。
  • 2. 確認新的程式碼包含完整的 <section id="about" …> 結構。
  • 3. 在本檔案中，找到「SECTION:ABOUT-START」與「SECTION:ABOUT-END」註解。
  • 4. 將兩個註解中間「整段 section」刪除，改貼上新的 section 程式碼。

小提醒：CSS 部分也可以配合使用「CSS:ABOUT」插槽，透過 class 選擇器覆寫這個區塊的排版與樣式，請避免使用 ID 選擇器。
